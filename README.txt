TASK DISTRIBUTION SYSTEM
-Sistemde 3 tane rol oluşturuldu.
    --Personel rolü sadece task ekleyebilmektedir.
    --Analist rolü eklenen taskları değerlendirip, taskın zorluk derecesini belirlemektedir.
    --Admin kullanıcı ekleme ve task atama işlemi yapabilir.

İŞLEYİŞ
1-Personel, giriş yaptıktan sonra üst panelde bulunan kullanıcı task ekleme ekranına  yönlendirilir.Burada task ekleme sayfasından task ekleyebilmektedir.
2-Analist de sisteme giriş yaptıktan sonra analist panelinde personelin girdiği taskları görür ve her birine bir zorluk seviyesi atar.
3-Admin kendisine açılan admin panelinde 2 iş yapabilir.Birincisi Task Atama İşlemleri butonuna basarak zorluk seviyesi belirlenmiş olan taskları şirkette
bulunan yazılımcılara atama işlemi yapabilir.İkinci olarak da Kullanıcı Listesi butonundan kullanıcı yönetim sayfasına gider ve burada sisteme istediği rolde
kullanıcı atayabilir.
